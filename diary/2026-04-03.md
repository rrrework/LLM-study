# 1、dropout 的作用是什么？
dropout是一种正则化手段，在训练过程中随机丢弃一部分神经元，防止网络在学习中过度依赖某一特征，降低过拟合的风险。
# 2、batch normalization 有什么作用？
对中间层激活的归一化，使得训练时更稳定，收敛更快，同时也有一定的正则化效果。平常还经常用到LayerNorm。
# 3、PyTorch 中张量的基本操作有哪些？
张量的基本操作有创建张量（torch.tensor）、查看张量的属性（shape、size）、索引和切片、以及维度的变换（view、reshape、permute等）、对张量的加减乘除、矩阵相乘等操作。
# 4、你做的 RAG 项目中，为什么用 Chroma
一是因为Chroma这个向量数据库是开源的，二是因为Choroma它适合本地原型开发，集成简单、上手也快，和Python生态的兼容性比较好。对于我的科研资料问答助手而言，本地单机的知识库已经能满足需求，不需要一开始就引入更重的分布式向量数据库，等数据量和并发起来后，再考虑这类方案更合理。
# 5、除了 Chroma，你还了解哪些向量数据库？
Milvus，开源分布式的向量数据库，主打高性能和大规模（可到十亿级向量），有 GPU 加速和多种 ANN 索引，适合企业级 RAG/推荐系统。
Weaviate：开源、云原生，支持向量 + 结构化/倒排索引混合搜索，适合内容检索、知识库搜索等场景。
Qdrant：开源（Rust 实现），强调性能/成本比和过滤查询能力，适合自托管中小规模到中大型 RAG 应用。
